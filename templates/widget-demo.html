Embeddable widget

Add face verification to any web app with one script tag and a verify-role API key. Register your site's origin in the admin console (Tenant settings → CORS) so the browser calls are allowed.

Embed

<script src="{{ '' }}/widget.js"></script>
<face-verify base="THIS_HOST" api-key="fk_your_verify_key"></face-verify>

<script>
  document.querySelector('face-verify')
    .addEventListener('result', e => {
      if (e.detail.success) {
        // e.detail is the signed API result
        console.log('Welcome', e.detail.user_id);
      }
    });
</script>

Optional attributes: user-id (1:1 verify a specific person), accent (brand colour), button-text.
